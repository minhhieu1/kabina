# Kabina

## Introduction
### Tools and techniques:
#### Frontend: 
Angular 7 + Bootstrap + HTML5 + CSS + TypeScript 

IDE: Visual Studio Code
#### Backend: 
RESTFull Spring Boot + JPA 

IDE: Eclipse
#### Database: 
mySQL
#### Security:
JSON Web Token

### Requirement:
  Validate user.

  Book or release a cabinate.

  Approve or reject booking.

  Search cabinate.

  User history.

  Report to Admin.
